## 🚀 使用方式

### 方式一：通过设置面板（推荐）

1. 点击顶部导航栏的"布局配置"按钮
2. 在打开的抽屉中找到"主题预设"区域
3. 点击任意预设即可应用

### 方式二：代码调用

```typescript
// 获取设置组件引用
const setingsRef = ref();

// 打开设置面板
setingsRef.value.openDrawer();
```

    menuBar: '#ffffff',

    topBar: '#ffffff',
    topBarColor: '#475569'

## 📋 效果预览

设置页面现在包含以下部分（从上到下）：

1. **主题预设** - 7 个彩色圆点 + 名称，点击切换
2. **全局主题** - primary 颜色选择器、深色模式开关
3. **顶栏设置** - 顶栏背景色、字体颜色
4. **菜单设置** - 菜单背景色、字体颜色、高亮色
5. **分栏设置** - 分栏菜单相关设置
6. **界面设置** - 折叠、手风琴、固定 Header 等
7. **界面显示** - Logo、面包屑、标签页、页脚等
8. **其它设置** - 标签页风格、动画、布局切换
