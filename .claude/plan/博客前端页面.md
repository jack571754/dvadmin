## 任务概述

创建具备专业美感的个人博客前端页面，采用极简主义设计风格（Medium.com 风格）。

## 路由配置

| 路径 | 组件 | 说明 |
|------|------|------|
| `/` | Home.vue | 博客首页 - 文章列表 |
| `/article/:id` | ArticleDetail.vue | 文章详情页 |
| `/archive` | Archive.vue | 归档页 - 分类/标签 |
| `/about` | About.vue | 关于页 |

---

## API 对接

### 后端接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/blog/articles/` | GET | 获取文章列表 |
| `/api/blog/articles/{id}/` | GET | 获取文章详情 |
| `/api/blog/categories/` | GET | 获取分类列表 |
| `/api/blog/tags/` | GET | 获取标签列表 |
| `/api/blog/articles/hot/` | GET | 获取热门文章 |

### 请求拦截

- 自动添加 JWT Token（如果存在）
- 10 秒超时
- 统一错误处理

## 开发命令

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
# 访问 http://localhost:5173

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```
