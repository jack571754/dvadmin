# 博客页面头部对齐修复

## 任务描述
修复 About.vue 和 Archive.vue 页面的头部元素对齐问题

## 问题分析
- **Archive.vue**: `.page-header` 使用 `text-align: center` 无法正确居中装饰元素
- **About.vue**: `.header-decoration` 使用 `width: 100%` 导致子元素无法正确对齐

## 解决方案
使用 flexbox 布局确保所有头部元素居中对齐

## 执行步骤

### 1. 修复 Archive.vue
**文件**: `blog-frontend/src/views/Archive.vue`

**修改内容**:
```css
/* 修改前 */
.page-header {
  text-align: center;
  padding: var(--space-3xl) var(--space-lg) var(--space-2xl);
}

.header-decoration {
  display: flex;
  align-items: center;
  justify-content: center;
  margin: 0 auto var(--space-md);
  width: fit-content;
}

/* 修改后 */
.page-header {
  display: flex;
  flex-direction: column;
  align-items: center;
  padding: var(--space-3xl) var(--space-lg) var(--space-2xl);
}

.header-decoration {
  margin-bottom: var(--space-md);
}
```

### 2. 修复 About.vue
**文件**: `blog-frontend/src/views/About.vue`

**修改内容**:
```css
/* 简化样式，移除冗余的 position 和 width */
.about-header {
  padding: var(--space-20) var(--space-6);
  display: flex;
  flex-direction: column;
  align-items: center;
}

.header-inner {
  max-width: var(--container-lg);
  display: flex;
  flex-direction: column;
  align-items: center;
}

.header-decoration {
  margin-bottom: var(--space-6);
}
```

## 预期结果
- 装饰元素（圆点）与标题垂直居中对齐
- 所有头部元素水平居中
- 保持响应式布局正常工作
