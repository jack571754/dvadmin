## 配色方案

### 柔和清新马卡龙色系

| 色彩 | 色值 | 色相 | 说明 |
|------|------|------|------|
| **主色调** | #a78bfa | 柔和紫蓝色 | 温柔、友好 |
| **成功色** | #4ade80 | 清新薄荷绿 | 积极、健康 |

### 配色特点

- **低饱和度**：减少视觉疲劳，适合长时间使用
- **高明度**：界面更明亮通透
- **柔和阴影**：使用轻量阴影，增强层次感
- **圆润圆角**：更大的圆角值，更亲和

## 使用方式

### 启动项目

```bash
cd django-vue3-admin-master/web
yarn run dev
```

### 使用设计令牌

在组件中使用设计令牌：

```vue
<template>
  <div class="my-component">
    <!-- 使用 UnoCSS 快捷类 -->
    <div class="card-base card-hover">
      <button class="btn-base">按钮</button>
    </div>
  </div>
</template>

<style lang="scss" scoped>
.my-component {
  // 使用设计令牌变量
  padding: var(--spacing-lg);
  border-radius: var(--radius-md);
  background: hsl(var(--color-bg-card));
  color: hsl(var(--color-fg-base));
  box-shadow: var(--shadow-sm);
  transition: var(--transition-all);
}
</style>
```

### 深色模式切换

#### 方式一：使用组件

```vue
<template>
  <ThemeSwitch />
</template>

<script setup>
import ThemeSwitch from '/@/layout/navBars/breadcrumb/themeSwitch.vue';
</script>
```

#### 方式二：编程方式

```typescript
import { useThemeConfig } from '/@/stores/themeConfig';

const themeConfigStore = useThemeConfig();

// 切换深色模式
themeConfigStore.setThemeConfig({
  themeConfig: {
    ...themeConfigStore.themeConfig,
    isIsDark: true
  }
});
```

### 与 DVAdmin 主题配置兼容

原有的主题配置功能完全兼容，使用方式不变：

```typescript
import { useThemeConfig } from '/@/stores/themeConfig';

const themeConfigStore = useThemeConfig();

// 更改主色调
themeConfigStore.setThemeConfig({
  themeConfig: {
    ...themeConfigStore.themeConfig,
    primary: '#67c23a'  // 会自动转换为 HSL 并应用
  }
});

// 更改顶栏颜色
themeConfigStore.setThemeConfig({
  themeConfig: {
    ...themeConfigStore.themeConfig,
    topBar: '#f5f5f5',
    topBarColor: '#303133'
  }
});
```

## 自定义主题

### 修改主色调

编辑 `src/theme/tokens.scss`：

```scss
:root {
  /* 将默认的蓝色 #409eff 改为其他颜色 */
  --color-primary: 142 71% 45%; /* 绿色 */
}
```

### 添加自定义快捷类

编辑 `uno.config.ts`：

```typescript
shortcuts: {
  // 添加新的快捷类
  'custom-card': 'bg-white dark:bg-gray-800 rounded-lg shadow-md p-4',
}
```

## 注意事项

1. **兼容性**：新的样式系统完全兼容 DVAdmin 原有的主题配置
2. **渐进增强**：现有页面会自动应用新样式，无需修改
3. **深色模式**：使用 `.dark` 类名，可全局或局部应用
4. **性能**：UnoCSS 按需生成，不会增加包体积
